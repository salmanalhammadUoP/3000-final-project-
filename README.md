my lawyer 
a website that connects clients with legal professionals. 
uses fire base, stripe, html, css, java script
live on https://mylawyer-dfe9d.web.app

features 
role based autehntication 
lawyer discovery, filter and sort
ratings and reviews
chat with lawyers
easy and secure payment using stripe
responsive design for all devices

how it works 
client journey 
1. Register and log in as a client
2. Browse lawyers  filter by specialisation, sort by rating or price
3. View a lawyers profile, read reviews, leave a rating
4. Start a consultation start a chat
5. Receive a payment request from the lawyer and pay securely using Stripe

Lawyer Journey
1. Register with bar licence, specialisation, and hourly rate
2. View all client enquiries sorted by most recent
3. Reply to clients in real time
4. Send an invoice using the bill client button then client pays through the platform
